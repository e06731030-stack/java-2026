## 📖 Descripción del Proyecto

Este proyecto fue desarrollado como parte del curso de **Desarrollo de Aplicaciones**, utilizando el lenguaje de programación **Java**.

El objetivo principal es fortalecer los conocimientos en programación orientada a objetos, estructuras de control, manejo de clases, métodos y desarrollo de aplicaciones prácticas.
